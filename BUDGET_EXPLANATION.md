# Зачем нужен бюджет в BankCentr?

## Назначение бюджета

**Бюджет** — это финансовый план, который помогает контролировать расходы и управлять личными финансами более эффективно.

## Основные функции бюджета:

### 1. **Контроль расходов**
Бюджет устанавливает лимит на общие расходы за период. Система сравнивает фактические траты с установленным бюджетом и показывает:
- Сколько потрачено из бюджета
- Сколько осталось до лимита
- Превышен ли бюджет

### 2. **AI-аналитика соблюдения бюджета**
При запросе аналитики типа `BUDGET_COMPLIANCE` система:
- Сравнивает фактические расходы с установленным бюджетом
- Определяет категории, где превышен бюджет
- Выявляет категории, где есть резерв средств
- Предоставляет рекомендации по улучшению контроля бюджета

### 3. **Финансовое планирование**
Бюджет помогает:
- Планировать расходы заранее
- Избегать импульсивных покупок
- Достигать финансовых целей
- Копить деньги на важные покупки

### 4. **Уведомления и предупреждения**
Система может отправлять уведомления:
- Когда расходы приближаются к лимиту бюджета
- Когда бюджет превышен
- О рекомендациях по оптимизации расходов

## Как это работает:

1. **Установка бюджета**: Пользователь устанавливает месячный или другой периодный бюджет через `updateBudget` mutation
2. **Отслеживание транзакций**: Система автоматически отслеживает все транзакции со статусом `COMPLETED`
3. **Анализ**: При запросе аналитики система сравнивает фактические расходы с бюджетом
4. **Рекомендации**: AI анализирует данные и предоставляет персональные рекомендации

## Пример использования:

```graphql
# Установка бюджета на 50000 рублей
mutation {
  updateBudget(input: { amount: 50000 }) {
    id
    amount
  }
}

# Запрос аналитики соблюдения бюджета
mutation {
  createAnalyticsTask(input: {
    type: BUDGET_COMPLIANCE
    dateFrom: "2024-12-01T00:00:00Z"
    dateTo: "2024-12-31T23:59:59Z"
  }) {
    id
    status
  }
}
```

## Преимущества использования бюджета:

✅ **Контроль**: Всегда знаете, сколько можете потратить  
✅ **Дисциплина**: Помогает избегать лишних трат  
✅ **Планирование**: Легче планировать крупные покупки  
✅ **Аналитика**: Получаете детальный анализ своих расходов  
✅ **Рекомендации**: AI предлагает способы оптимизации финансов
